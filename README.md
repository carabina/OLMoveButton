# OLMoveButton
### AssistiveTouch &amp; move Button  可以移动的按钮

###两种不同效果只需设置如下属性就可
```
/**
 *  拖拽后是否自动返回两侧
 */
@property (nonatomic, assign) BOOL isAutoBack;
```
